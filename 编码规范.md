1. 文件注释

```
/**
 * @file Describe the file
 * @author auther-name(mail-name@domain.com)
 *         auther-name(mail-name@domain.com)
 */
```


2. 类注释

> 建议使用@class标记类或构造函数, 使用@extends标记类的继承信息
```
/**
 * 类描述
 *
 * @class
 * @extends Developer
 */
var Fronteer = function () {
    Develop.call(this);

    /**
     * 属性描述
     *
     * @type {string}
     * @private
     */
     this.level = 'T12';
};


/**
 * 方法描述
 *
 * @private
 * return {string} 返回值描述
 */
```


3. 函数/方法注释

```
/**
 * 函数描述
 *
 * @param {string} p1 参数1的说明
 * @param {string} p2 参数2的说明
 * @param {number=} p3 参数3的说明
 * @return {Object} 返回值描述
 */
```

4. 事件注释

```
/**
 * 值变更时触发
 *
 * @event Select#change
 * @param {Object} e e描述
 * @param {string} e.before e.before描述
 * @param {string} e.after e.after描述
 */

this.fire(
    'change',
    {
        before: 'foo',
        after: 'bar'
    }
);
```
