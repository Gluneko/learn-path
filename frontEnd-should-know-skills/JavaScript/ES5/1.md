// storage事件监听增加，修改，删除的动作。
function sendMsg(text) {
	window.localStorage.setItem('msg', text);
}

window.addEventListener('storage', function(evt) {
	if(evt.key == 'msg') {
		console.log(newValue);
	}

## 原生AJAX

xhr对象的5个状态：

* 0, 未初始化，还未调用send()
* 1, 载入，已调用send()
* 2, 载入完成，send()执行完成
* 3, 正在解析响应的内容
* 4，完成解析
